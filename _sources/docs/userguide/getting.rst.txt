Getting the |project_name|
==========================

There are two ways to get |project_name|:

Download
--------

For the direct download, select the "Source code (zip)" option on the following links:

* Latest release from the `github repository <https://github.com/GeoscienceAustralia/BAL/releases/latest>`_
* Earlier releases are also `available <https://github.com/GeoscienceAustralia/BAL/releases>`_


Clone the repository
--------------------

You can also clone the repository, which will enable you to easily
keep up to date with any updates to the code. Using HTTPS::
    
    $ git clone https://github.com/GeoscienceAustralia/bal.git

Using SSH::

    $ git clone git@github.com:GeoscienceAustralia/bal.git

Note that you will not be able to push changes back to the main
|project_name| repository. If you want to make changes, follow the
guidelines in :ref:`contributing`.
