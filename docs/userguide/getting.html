Getting the Bushfire Attack Level Toolbox¶

There are two ways to get Bushfire Attack Level Toolbox:

Download¶

For the direct download, select the “Source code (zip)” option on the following links:

  • Latest release from the github repository
  • Earlier releases are also available

Clone the repository¶

You can also clone the repository, which will enable you to easily keep up to date with any updates to the code. Using HTTPS:

$ git clone https://github.com/GeoscienceAustralia/bal.git

Using SSH:

$ git clone git@github.com:GeoscienceAustralia/bal.git

Note that you will not be able to push changes back to the main Bushfire Attack Level Toolbox repository. If you want to make changes, follow the guidelines in Contributing code.
